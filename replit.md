# High Speed Lynx

A high-speed platformer game built with PHP, JavaScript, and HTML5 Canvas.

## Project Structure
- `index.php`: Main game page.
- `functions.php`: Reusable PHP logic.
- `about.php`: Game rules and information.
- `leaderboard.php`: Player rankings.
- `js/game.js`: Core game mechanics.
- `css/style.css`: Visual styling.
- `data/gamePlay.json`: Persistence for scores and player data.

## Features
- Fast-paced lynx movement.
- Procedural level generation (planned).
- Spin Charge mechanic.
- JSON-based leaderboard.
